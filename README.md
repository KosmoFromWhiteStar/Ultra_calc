# Ultra_calc


**Ultra_calc** — это легковесный ПЭТ-проект, представляющий собой C-библиотеку для математических расчётов и вычисления различных функций. Проект спроектирован с упором на модульность, простоту интеграции и производительность кода на языке C.

## ✨ Особенности (Features)
* **Чистый C:** Полная совместимость со стандартами языка C.
* **Модульная структура:** Чёткое разделение интерфейса (`HEADER.H`) и реализации (`HEADER.C`).
* **Автоматизация сборки:** Быстрая конфигурация и сборка проекта с помощью CMake.

## 🛠 Стек технологий
* **Язык программирования:** C
* **Система сборки:** CMake

## 📂 Структура проекта
* `main.c` — Демонстрационный файл для проверки работы функций.
* `HEADER.H` — Заголовочный файл с объявлениями математических функций и макросов.
* `HEADER.C` — Исходный код с логикой и реализацией функций расчёта.
* `CMakeLists.txt` — Скрипт конфигурации сборки CMake.

## 🚀 Быстрый старт и сборка

Для сборки и запуска проекта вам понадобятся компилятор (GCC / Clang) и установленный [CMake](https://cmake.org).

### 1. Клонирование репозитория
```bash
git clone https://github.com
cd Ultra_calc
```

### 2. Сборка проекта
Используйте стандартный рабочий процесс CMake для генерации файлов сборки и компиляции:
```bash
# Создание директории для сборки
mkdir build
cd build

# Конфигурация проекта
cmake ..

# Компиляция исполняемого файла
cmake --build .
```

### 3. Запуск демо-программы
После успешной сборки внутри папки `build` появится исполняемый файл. Запустите его:
```bash
./Ultra_calc
```

## 📦 Интеграция в другие проекты
Чтобы использовать эти расчёты в своём коде, достаточно скопировать файлы `HEADER.H` и `HEADER.C` в ваш проект и подключить заголовок:

```c
#include "HEADER.H"

int main() {
    // Вызов математических функций из библиотеки Ultra_calc
    return 0;
}
```
